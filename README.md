# SAT-Superior

Project: SAT Superior

What it does:
An AI chatbot that helps students study for the SAT by answering questions, explaining concepts, generating SAT-style practice questions, and providing study tips.

Features:
- Answers SAT Math, Reading, and Writing questions
- Explains solutions step-by-step
- Generates practice questions
- Provides grammar rules, formulas, and study strategies
- Available 24/7 through an AI chatbot

Stack:
Python, OpenAI API, Streamlit
